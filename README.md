# vehicles_add_app

Este proyecto busca iniciar en el proceso de creaciones de aplicaciones web con Python.

Se ha creado una app web que muestra 2 gráficos de diferente tipo por medio de la selección de sus casillas correspondientes, los gráficos están basados en una base de datos dada acerca de anuncios de vehiculos y sus diferentes especificaciones.
La app ha sido desplegada en la plataforma Render.com y para hacerlo se ha utilizado Streamlit.

Esta aplicación tiene como objetivo aprender a hacer desde cero una app, puede encontrarse en: https://vehicles-add-app.onrender.com
